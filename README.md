# IPL-ANALYSIS-_Tableau_Dashboard
The IPL Analysis Dashboard provides a comprehensive view of Indian Premier League (IPL) data, offering insights into team and player performance across multiple seasons. It helps users explore match statistics, player contributions, and trends through interactive visualizations and filters.

Key Features

1. Match Overview: Displays total matches played, wins, losses, and team standings.

2.Player Performance: Highlights top run-scorers, wicket-takers, and player consistency over seasons.

3. Team Insights: Compares teams based on batting averages, strike rates, economy rates, and win percentages.

4. Venue Analysis: Shows performance trends by stadium or city.

5.Seasonal Trends: Visualizes year-by-year statistics to identify dominant teams and emerging players.

6.Interactive Filters: Allows users to filter by team, player, venue, or season for deeper insights.
